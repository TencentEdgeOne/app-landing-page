# APP 落地页模板
本模板是基于 Next.js 静态生成和 Tailwind CSS 实现的 App 落地页，外观设计来自[
Zelal Hossain](https://www.figma.com/community/file/995026220622307527)。

## 本地开发
安装依赖：`npm install`
本地调试：`npm run dev`

## 样式主题
在 `/src/styles/globals.css` 可以修改样式颜色。
